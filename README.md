### Tools:

- Raspberry Pi 3
- [PIR sensor](https://learn.adafruit.com/pir-passive-infrared-proximity-motion-sensor/overview)
---
- Express
- Socket.IO
- Johnny Five
- Vue.js

### Start

- setup Raspberry PI w/ PIR [raspi-io](https://github.com/nebrius/raspi-io)
- npm install
- npm start
- open url: raspberry-ip:8080
